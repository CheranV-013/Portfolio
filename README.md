# React Portfolio Website

A modern, clean, and responsive portfolio website built for an Artificial Intelligence and Machine Learning student. The site highlights education, certifications, achievements, projects, and contact details with a light professional theme.

## Technologies Used
- React (Vite)
- Tailwind CSS
- React Icons

## Setup Instructions
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
